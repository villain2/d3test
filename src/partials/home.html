Image Drawing and Mapping

A suite of interactive applications to be integrated into the CEWidgets as a more interactive way to teach math and geometry.

Connexus Link: https://www.connexus.com/issueAware/issue.aspx?idIssue=1176772

Create a new widget type that includes the following features/functions. The requirements for this widget are targeted to math-specific content. If similar requirements are requested later for other subject areas, that would be deemed a different widget type. Given the specific needs of math functions, it has been deemed necessary to break this out as its own widget type.
